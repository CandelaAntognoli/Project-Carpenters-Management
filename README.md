# Project Carpenters: Gestión Operativa y Financiera 🪵🗂️

![Project Carpenters Cover](img/project%20carpenters.png)

Este repositorio contiene una solución integral de gestión desarrollada en **Google Sheets** para una empresa de carpintería. El proyecto transforma procesos manuales en un flujo de trabajo automatizado y basado en datos.

## 🚀 Impacto del Proyecto
*   **Ahorro de tiempo:** Reducción de 10 horas semanales en administración manual.
*   **Precisión:** Disminución del 90% en errores de facturación mediante automatización.
*   **Visibilidad:** Seguimiento en tiempo real de márgenes de ganancia y estados de órdenes.

## 🛠️ Soluciones Implementadas
*   **Dashboard Interactivo:** Panel de control para monitorear tareas de alta prioridad, órdenes pendientes de entrega y facturación vencida.
*   **Sistema de Facturación (Invoice):** Generador dinámico de facturas de clientes basado en la base de datos de órdenes.
*   **Módulo de Analytics:** Gráficos de distribución de ganancias por producto y seguimiento de horas invertidas vs. presupuestadas por tarea.
*   **Gestión de Tareas:** Sistema de seguimiento con indicadores de prioridad (High, Med, Low) y porcentajes de cumplimiento.

## 📊 Habilidades Técnicas
*   **Automatización:** Uso de fórmulas lógicas complejas y validación de datos.
*   **Project Management:** Estructuración de flujos de trabajo operativos.
*   **Data Integrity:** Normalización de registros de clientes y productos.
